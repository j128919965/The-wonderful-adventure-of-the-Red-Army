操作说明：

wasd：上下左右移动光标
j：取消操作/返回上级界面
k：选中单位或装备/确认
i：展示当前位置的角色/装备信息

游戏说明：

该游戏为战棋游戏，我方操作蓝色的单位。
每个角色有不同的生命值，攻击力，防御力和闪避率，以及不同的移动范围。
玩家可以操作黑色光标，选择不同的角色。每个角色在一回合内只可以行动一次。
角色可以移动，可以对身边的敌人发起攻击，可以装备物品，可以使用药品。
角色战斗可以获得经验值，经验值达到100会升级，升级可以获得随机的属性提升。
人物站在不同的地形上，会获得不同的增益效果。
角色生命值降到0之后则阵亡。
角色行动之后会进入待机状态。当某一方的所有角色都进入待机状态时，将进入下一回合。
我方角色全部阵亡后，游戏结束。我方任意角色击败 敌方头目 胡汉三之后
站在城门位置，将会获得游戏胜利。
